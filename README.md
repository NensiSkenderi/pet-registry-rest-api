# Pet Registry Rest Api

## About The Project

A Spring Boot project to manage pets, pet owners and visits to the vet.

## Technologies used: 

  * Spring Boot
  * REST
  * Jpa (Hibernate)
  * MySQL
  * Spring Security Oauth2
  * Model Mapper
    
Also, in this projet I have used a simple Custom Error Handling
